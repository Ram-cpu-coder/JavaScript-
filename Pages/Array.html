Arrays in JavaScript

Array is a special variable that can hold multiple values under the single name.

const cars = [ "Saab", "Volvo", "BMW" ];

Why use Array?

Lets take an example, if we need to have a list of 3 cars name, it will look like this:

let car1 = ["Saab"];

let car2 = ["Volvo"];

let car3 = ["BMW"];

But, what if we want to loop through the cars and find a specific one and we don't have just 3 cars but 300?

The solution is an Array!

An array can hold multiple values under a single name and those values can be accessed by using the index number.

Syntax

It is common way to declare an array using const keyword.

const array_name = ["Item1", "Item2", "Item3", ... ];

A Little Bit of History

In the early days of the internet, webpages were pretty plain and didn't change after you opened them. In 1995, a company called Netscape wanted to make websites more exciting. They created JavaScript to help make webpages come to life!

JavaScript was originally called "LiveScript," but then it was renamed to JavaScript to ride on the popularity of another language called Java, even though they're not the same.

Why is it Important?

JavaScript is essential for modern web development. It helps make the internet interactive, allowing us to play games, send messages, and do so much more!
