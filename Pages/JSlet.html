JavaScript Programs

Computer program is the set of the "instructions" to be executed by a computer.

Changing the innerHTML property of an innerHTML is a common way to display a data in HTML.

And in a programming language, these instructions are called statement.

A JavaScript program is a list of those programming statements.

In HTML, JavaScript programs are executed by the web browser.

JavaScript Statements

JavaScript statements are composed of:

Values, Operators, Expressions, Keywords and Comments.

Example

Most JavaScript programs contain many JavaScript statements.

The Statements are executed, one by one, in the same order as they are written.

JavaScript programs (and JavaScript Statements) are often called JavaScript code.

Semicolons ;

Semicolons separate the JavaScript Statements.

We need to add Semicolon at the end of each statement.

When separted by semicolons, multiple Statements on one line are allowed.

On the web, we might see the examples without semicolons.

Ending statement with the semicolons is not required, but highly recommended

JavaScript White Space

In JavaScript, extra spaces are ignored.

So, its better to make our code look good, clear and readable.


It is the best idea to put spaces around the Operators like =, +, -, *, /.

JavaScript Code Blocks

JavaScript statements can be grouped together in the block inside the curly bracket {...}

The purpose JavaScript code block is to define the statements that is to be executed together.

Common place to find the code block is in JavaScript Functions.

JavaScript Keywords

JavaScript Statements often start with the Keywords to identify the JavaScript action to be performed.

Some of the common Keywords are:

Keyword Description
var Declares the variable
let Declares a block variable
const Declares a block constant
if Marks a block of statements to be executed on a condition
switch Marks a block of statements to be executed in different cases
for Marks a block of statements to be executed in a loop
function Declares a function
return Exits a function
try Implements error handling to a block of statements
