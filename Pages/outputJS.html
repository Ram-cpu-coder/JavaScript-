JavaScript Output

JavaScript Display Possibilities

JavaScript can display output in various ways. They are:

  • By using innerHTML.
  • By using window.write().
  • By using console.log.
  • By using window.alert().


    • Using innerHTML

    To access HTML element, document.getElementById(id) is used.
    Here, id attribute defines the HTML element.

    Example

    Changing the innerHTML property of an innerHTML is a common way to display a data in HTML.

    Using window.write()

    This is used for the convenience of testing.

    Example

    Using document.write() after an HTML document is loaded, it will delete all the existing HTML.

    This is to be used only for the testing purpose.

    Using console.log()

    This method, console.log() is used for debugging process in browser to display data.

    Using window.alert()

    window.alert() is used to display the alert box on the top part of browser.

    The keyword window can be skipped.

    In JavaScript, window is optional. It is because window is the global scope object.

    It means by default, variables, properties, and methods belong to the window object.

    Using window.print()

    In JavaScript, there are not any print objects or print methods.

    We cannot access the output devices from JavaScript

    Only exception is that we can call window.print() method in the browser to print the content of the current window.
