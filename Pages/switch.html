JavaScript Programs

If condition is the statement that executes the code under the true condition of the given condition.

We can take the example of the pedestrian that is crossing the road.

The person will check the light, if green for pedestrian, check left and right for the upcoming vehicles and if none, then cross the road.

This If condition needs to have condition which gives boolean value, true or false.

Example

In if condition, we need to know the truthy and falsy values.

Truthy: true, positive number, negative number, strings,

Falsy: false, zero, NaN, "", null, undefined

Sometimes we need to change the value to false and sometimes to true.

It can be done using '!'.
